올리브 카레

올리브를 고기대신 활용하면 씹히는 맛, 감칠 맛이 있어서 좋아요. 속이 편안해지는 것도 느낄 수 있죠.

2인분

씨 없는 올리브 15개 양파 1/2개 다진 마늘 2큰술 시판 카레, 물 소금, 후추 약간 꿀 1작은술

고명채소

다양한 올리브 10개 브로콜리 30g 방울 토마토 4개 그린빈스 30g 연근 40g 단호박 40g

올리브 카레 만들기

1. 씨 없는 올리브와 양파는 굵게 다져요. 2. 팬에 올리브오일을 두르고 양파와 올리브를 볶아주세요. 3. 다진 마늘을 추가해서 다시 볶다가 물, 카레를 넣어주세요. 4. 좋아하는 농도까지 끓이다가 꿀을 넣고 소금, 후추 간을 해주세요. *조리팁: 마지막에 꿀을 한 작은술 넣으면 카레가 더 부드러워지면서 맛있어요!

플레이팅 팁

채소를 담을 때는 다양한 채소를 섞어서 입체감 있게 담아주세요. 올리브도 품종을 섞어 올려주면 보기에도 좋고 맛도 풍요로워져요.

const vTitle = document.querySelectorAll(".view-title ul li");
const vCode = document.querySelectorAll(".view-cont > div");

vTitle.forEach((element, index) => {
    element.addEventListener("click", function(){

        vTitle.forEach((el)=>{
            el.classList.remove("active");
        });
        element.classList.add("active");

        vCode.forEach((el)=>{
            el.style.display = "none";
        });
        vCode[index].style.display = "block";
    });
});
